# Umich Reflection Rooms Map

## Overview
The **Umich Reflection Rooms Map** is a project designed to provide a comprehensive map of reflection rooms at the University of Michigan. These rooms offer a serene environment for students and faculty to reflect, meditate, or study quietly.

## Features
- Interactive map displaying the locations of reflection rooms.
- Search functionality to find rooms by name or location.
- Detailed information for each room, including capacity and amenities.

## Installation
To get started with the Umich Reflection Rooms Map, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/farhia96/umich-reflection-rooms-map.git
   ```
2. Navigate to the project directory:
   ```bash
   cd umich-reflection-rooms-map
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```

## Usage
Once the installation is complete, you can start the application with:
```bash
npm start
```

Open your browser and navigate to `http://localhost:3000` to view the map.
